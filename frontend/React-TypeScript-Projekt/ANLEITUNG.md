# Backend

1. Navigiere im Terminal in den Ordner backend: `cd React-TypeScript-Projekt/backend`
2. Zum updaten der Packages führe einmal `npm install` aus
3. Zum Starten: `npm run start`

# Frontend

1. Navigiere im Terminal in den Ordner frontend: `cd React-TypeScript-Projekt/frontend`
2. Zum updaten der Packages führe einmal `npm install` aus
3. Zum Starten: `npm run dev`
