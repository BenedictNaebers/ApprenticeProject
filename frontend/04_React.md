## React
React ist eine JavaScript-Bibliothek, die es Entwicklern ermöglicht, Benutzeroberflächen (UIs) zu erstellen.

Es basiert auf einem Konzept namens "komponentenbasierte Entwicklung", was bedeutet, dass Entwickler UIs als unabhängige, wiederverwendbare Komponenten erstellen können. Diese Komponenten können dann zusammengesetzt werden, um komplexe UIs zu erstellen.

Eine wichtige Eigenschaft von React ist seine Fähigkeit, den Zustand von UI-Komponenten zu verfolgen und automatisch zu aktualisieren, wenn dieser Zustand geändert wird. Dadurch kann React dynamische UIs erstellen, die auf Benutzerinteraktionen reagieren.

React ist auch bekannt für seine "virtuelle DOM"-Funktionalität, die es ihm ermöglicht, Änderungen an der Benutzeroberfläche schnell und effizient zu verarbeiten und die Anzahl der erforderlichen Browser-Updates zu minimieren.

## JS vs. JSX
Nun zum Unterschied zwischen JS und JSX: 
JavaScript (JS) kennst du bereits. 
JSX hingegen ist eine Erweiterung von JS, die es dir ermöglicht, HTML-ähnlichen Code direkt in deinen JS-Code einzufügen. 
Das bedeutet, dass du mit JSX einfacher und schneller Komponenten erstellen kannst, da du HTML-Elemente und JS-Code in einer Datei zusammenfügen kannst.
